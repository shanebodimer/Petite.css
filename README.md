# Petite.css

Petite.css is a lightweight, customizable CSS framework.

## Quick Start

Here's how to get started with Petite:

* Download ``petite.css`` and ``normalize.css``
* Include the files in your page header

or

* [Download the quick-start zip](s.zip)
